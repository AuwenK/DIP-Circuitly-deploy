### Interactive Circuit Analysis Learning Platform for NTU EEE Students
Impulse is a gamified, web-based educational tool designed to help Electrical & Electronic Engineering students master the complexities of circuit analysis. By moving away from static textbooks, Impulse provides a dynamic environment where students can test their knowledge of nodal analysis, mesh analysis, and complex circuit theorems in real-time.

#### 🚀 Key Features
##### 1. Interactive Learning Dashboard
A centralized hub where students can select from 8 core EEE modules, including:
- Fundamentals & Theorems
- Energy Storage Elements
- Transient & Steady-State
- Ideal Op-Amps
- Laplace Transforms
- Network Functions
- DC vs. AC
- Three-Phase Circuits

##### 2. Adaptive Question Engine
Dynamic problem-solving interface that presents circuit diagrams and multi-step calculation challenges. It supports complex mathematical notation (LaTeX) to ensure academic rigor.

##### 3. Comprehensive Module Review
A feedback-driven "post-game" analysis. If a student fails a module, the system provides detailed, step-by-step explanations for the correct answers, helping to bridge knowledge gaps immediately.
<img width="1210" height="1228" alt="image" src="https://github.com/user-attachments/assets/8d8b9aaa-1a1e-4504-ac21-81ff71c6eed3" />

##### 4. Personal Proficiency Radar
A data-driven visualization tool (Radar Chart) that tracks a user's XP and time spent across different topics. This allows students to identify their weak points at a glance and focus their revision effectively.
